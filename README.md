# SL FloodWatch – Sierra Leone Flood Monitoring System

## Overview

SL FloodWatch is a responsive web-based flood monitoring and reporting platform designed to help communities, emergency responders, and government agencies monitor flood risks across Sierra Leone.

The system provides real-time flood status updates, emergency alerts, interactive risk visualization, and a community reporting mechanism that enables citizens to report flood incidents quickly.

This project was developed as a Digital Public Good (DPG) concept for disaster preparedness and emergency response.

---

## Project Objectives

The primary objectives of SL FloodWatch are:

* Monitor flood-prone areas across Sierra Leone.
* Display water-level and rainfall information in an easy-to-understand format.
* Provide emergency alerts to affected communities.
* Allow citizens to report flood incidents.
* Improve disaster preparedness and response coordination.
* Demonstrate the use of web technologies in environmental monitoring.

---

## Features

### 1. Real-Time Monitoring Dashboard

* Displays water levels for monitored zones.
* Uses color-coded risk indicators:

  * 🔴 Critical
  * 🟡 Warning
  * 🟢 Normal
* Animated progress bars visualize flood severity.

### 2. Emergency Alerts System

* Shows latest flood warnings and evacuation notices.
* Includes timestamps and detailed descriptions.
* Highlights high-risk areas.

### 3. Interactive Risk Map

* Custom SVG map of Sierra Leone.
* Displays flood hotspots visually.
* Includes animated alert markers.
* Provides a risk legend for users.

### 4. Community Reporting System

Users can submit:

* Name
* Phone number
* Affected district
* Severity level
* Flood description

Submitted reports are acknowledged immediately through the interface.

### 5. Responsive Design

* Desktop friendly
* Tablet friendly
* Mobile friendly
* Collapsible navigation menu for smaller screens

### 6. Animated User Experience

* Dynamic rain background
* Smooth scrolling navigation
* Animated water-level indicators
* Alert animations
* Interactive hover effects
